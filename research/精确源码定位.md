## 编译决策

当前主机 D 盘只余约 5.6 GB、C 盘只余约 1.3 GB，并且没有可用 WSL Linux 发行版。完整 OnePlus manifest 同步及内核输出通常远超该空间，因此不在本机强行同步半套源码。

工程已准备 GitHub Actions 完整构建工作流：

```text
.github/workflows/build_android_ko.yml
```

它会同步精确 manifest，执行 `sun perf` 官方构建，定位真实 `.config` 和 `Module.symvers`，再以相同输出编译 `delta_kernel.ko`，最终一并导出 ABI 证据。
